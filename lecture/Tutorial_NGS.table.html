  2. Download the Saccharomyces genome from SGD site

  3. Downlod the read data from SRA and convert it -- This step already done for you, folder is on server or you can run the download script when you get home (rerequires curl, sra toolkit, and for download speedup Aspera client

    • For Asperga, get the web client and find the ascp binary and install in your path. For example on OSX it installs in "/Applications/AsperaÂ Connect.app/Contents/Resources/ascp".

    • Download script to obtain all the data is here https://github.com/hyphaltip/CSHL_2012_NGS/blob/master/data/download.sh
